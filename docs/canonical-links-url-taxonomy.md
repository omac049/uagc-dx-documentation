# Canonical Links & URL Taxonomy

*This page is currently under development.*

Guidelines for URL structure and canonical link implementation on UAGC websites.

## URL Structure Standards

URLs should follow these conventions:

- Use lowercase letters
- Use hyphens to separate words
- Keep URLs short and descriptive
- Avoid special characters
- Use clear hierarchy (e.g., /programs/undergraduate/business/)

## Canonical Link Implementation

Canonical links help prevent duplicate content issues by specifying the preferred version of a page. Implementation guidelines:

- Add canonical tags to all pages
- Use absolute URLs in canonical tags
- Handle pagination correctly
- Address cross-domain duplicates
