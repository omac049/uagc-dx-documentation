# GTM Configuration & dataLayer Management

*This page is currently under development.*

Guidelines for Google Tag Manager implementation and dataLayer management on UAGC websites.

## GTM Configuration

The GTM configuration for UAGC websites includes:

- Account and container structure
- Tag implementation standards
- Trigger configuration
- Variable naming conventions
- User permissions and roles

## dataLayer Structure

The standard dataLayer structure includes:

- Page metadata
- User information (anonymized)
- Event tracking data
- Ecommerce data

## Implementation Guidelines

- Always use the dataLayer for passing data to GTM
- Follow naming conventions for consistency
- Document all custom events and variables
- Test tags in debug mode before publishing
