# Cookie Organization & Management

*This page is currently under development.*

Guidelines for cookie implementation and management on UAGC websites.

## Cookie Categories

UAGC websites use the following cookie categories:

1. **Essential** - Required for basic website functionality
2. **Functional** - Enhance user experience but not essential
3. **Analytics** - Collect anonymous usage data
4. **Marketing** - Used for targeted advertising and campaigns
