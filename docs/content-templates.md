# Content Templates

*This page is currently under development.*

Standardized templates for various content types on UAGC websites.

## Available Templates

- Program Page Template
- Faculty Profile Template
- News Article Template
- Event Page Template
- Landing Page Template

## Usage Guidelines

Templates should be used to maintain consistency across the website while allowing for necessary customization. Each template includes:

- Required sections and elements
- Recommended content length
- Placeholder text examples
- SEO recommendations
